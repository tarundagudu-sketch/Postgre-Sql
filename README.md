# Amazon Brazil Data Analysis (PostgreSQL)

##  Project Overview
This project performs a deep dive into Amazon Brazil's sales data to identify seasonal patterns, payment preferences, and customer segmentation. The analysis is divided into three comprehensive parts:
1. **Financial Standardization & Sales Patterns**
2. **Customer Behavior & Revenue Analysis**
3. **Growth Trends & Advanced Analytics**

##  Key Insights & Recommendations
For a full breakdown of the business recommendations and visual charts, please refer to the:

##  Technical Skills Demonstrated
* **Joins & Aggregations:** Combining customer, order, and payment tables.
* **Window Functions:** Using `RANK()` and `LAG()` for growth analysis.
* **CTE & Recursive Queries:** Calculating monthly cumulative sales.
* **Data Segmentation:** Categorizing customers into 'New', 'Returning', and 'Loyal' segments.
